# EduOnline (Student Project)

**Author:** Aida Kentay  
**Group:** SE-2428  
**Year:** 2025

## Overview
A multi-page responsive demo website for an online learning platform implemented with HTML, CSS, Bootstrap, JavaScript and jQuery.

## Pages
- index.html — Home / Hero
- courses.html — Courses (cards)
- progress.html — Progress (grades table) + contact form
- gallery.html — Image gallery with jQuery effects
- contact.html — Registration / file upload / calculator / live input

## How to run
Open `index.html` in a modern browser. No backend needed.

## Features implemented
- Responsive layout with Bootstrap & custom CSS
- Forms with HTML5 attributes and Bootstrap validation styles
- JS: form demo handling, calculator, live input, confirmation dialogs
- jQuery effects: gallery shuffle, fade transitions
