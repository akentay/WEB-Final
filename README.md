**Author:** Aida Kentay

---

## Project Description

Learnify is an interactive web platform for online learning, developed as a frontend development project. Users can browse courses, add them to favorites and cart, manage personal tasks, read blog articles, and learn about the platform. The project demonstrates proficiency in HTML, CSS, JavaScript, Bootstrap, and LocalStorage.

---

## Features

- **Home Page:**  
  - Popular courses displayed as cards  
  - Personal task list with add, mark as done, and delete functionality  
  - Brief information about the platform  

- **Courses Page:**  
  - Filter by level (Beginner, Intermediate, Advanced)  
  - Search courses by name  
  - Detailed course page with description  
  - Add courses to cart and favorites  

- **Blog Page:**  
  - List of posts with images and text preview  
  - Filter posts by tags  
  - View individual blog post with content and related posts  

- **About Page:**  
  - Project story, mission, and team  
  - Images and video (can be local video)  

- **User Profile:**  
  - View favorites and cart  
  - Simulated checkout via modal form  
  - Local login/logout  

- **Additional Features:**  
  - Tasks and favorites are stored in LocalStorage  
  - Fully responsive layout for mobile and tablet  
  - Uses Bootstrap 5 and Bootstrap Icons  

---

## Technologies

- **HTML5** — page structure  
- **CSS3** — styling and responsive layout  
- **JavaScript (Vanilla JS)** — app logic, routing, LocalStorage  
- **Bootstrap 5** — grid system, cards, modals, buttons  
- **LocalStorage** — store tasks, favorites, cart, and user session
